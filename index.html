Einstellungen ▼
i
Steuert die Zufälligkeit der Antworten.

0.0 = Deterministisch, wählt immer das wahrscheinlichste Wort. Gut für Fakten und Code.
0.5–0.7 = Ausgewogen. Guter Standard für die meisten Aufgaben.
1.0+ = Sehr kreativ und variabel. Kann zu unvorhersehbaren Ergebnissen führen.
0.7
Niedrig = fokussiert, hoch = kreativer
i
Begrenzt die maximale Länge der Antwort in Tokens (ca. 1 Token = 0.75 Wörter).

256–512 = Kurze, knappe Antworten.
1024–2048 = Ausführliche Erklärungen.
4096+ = Sehr lange Texte, Code oder Artikel. Hohe Werte brauchen mehr Zeit und Speicher.
2048
Maximale Länge der Antwort
i
Nucleus-Sampling: Das Modell wählt nur aus den Tokens, deren kumulierte Wahrscheinlichkeit diesen Wert erreicht.

0.1 = Nur die allerbesten Tokens. Sehr vorhersagbar.
0.9 = Breite Auswahl, natürlicherer Text.
1.0 = Alle Tokens möglich.

Tipp: Entweder Temperature oder Top-P anpassen, selten beides gleichzeitig.
0.9
Nucleus-Sampling: begrenzt die Token-Auswahl auf die wahrscheinlichsten
